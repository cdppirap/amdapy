Detailed Description

pdsdump.py script for querying PDS ASCII datasets, and converting them to netCDF.

Author
Alexandre Schulz

Query and convert PDS ASCII files. Usage : –dump : print contents of the dataset –convert2netcdf : convert file to netcdf -o, –output : output filename
